# UNSW-NB15 Dataset

## Download Instructions:
1. Go to https://research.unsw.edu.au/projects/unsw-nb15-dataset
2. Download the CSV files
3. Place them in this directory

- UNSW-NB15_1.csv
- UNSW-NB15_2.csv
- UNSW-NB15_3.csv
- UNSW-NB15_4.csv
