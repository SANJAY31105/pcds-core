# How to Create in Google Forms:

1. Go to https://forms.google.com
2. Click "+ Blank"
3. Title: "PCDS Validation Survey"
4. Add each question above
5. Set Q1-Q2 as multiple choice
6. Set Q3-Q6 as linear scale (1-5)
7. Set Q7-Q8 as multiple choice
8. Set Q9 as linear scale (0-10)
9. Set Q10-Q14 as short/paragraph text
10. Make Q12-Q14 optional
